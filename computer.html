HARDWARE COMPONENTS

CPU (CENTRAL PROCESSING UNIT)

Often referred to as the brain of the computer, the CPU performs calculations and executes instructions from programs. It processes data and controls other components

MOTHERBOARD

This is the main circuit board that connects all components of the computer. It houses the CPU, RAM, and provides connectors for other peripherals.

CPU FAN

The CPU fan cools the Central Processing Unit (CPU) by dissipating heat generated during operation.

GPU (Graphics Processing Unit)

This component is responsible for rendering images and video. It is especially important for gaming and graphic design.

RAM (Random Access Memory)

RAM is the short-term memory of the computer, temporarily storing data that the CPU needs while performing tasks. It is volatile, meaning it loses its data when the computer is turned off.

Power Supply Unit (PSU)

The PSU converts electrical power from an outlet into usable power for the internal components of the computer.

HDD (Hard Disk Drive)

A Hard Disk Drive (HDD) is an electro-mechanical data storage device that uses magnetic storage to store and retrieve digital data on one or more rapidly rotating platters

FAN

The primary function of a fan is to dissipate heat generated by components like the CPU and GPU, preventing overheating.

CASE

The case provides physical protection for components like the motherboard, CPU, GPU, hard drives, and power supply.
